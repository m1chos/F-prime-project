# Hardware setup

[Back to main page](../README.md)

### Components
* Raspberry Pi Pico
* MPU6050 via I2C
    * Pin 4: Serial Data Line (SDA)
    * Pin 5: Serial Clock Line (SCL)
* Red, Green and Blue LED
    * GPIO Pins
* 3x 220 Ohm resistors for the LEDs

> Refer to [Pico Serier](https://www.raspberrypi.com/documentation/microcontrollers/pico-series.html) and [RP2040 zero](https://www.waveshare.com/wiki/RP2040-Zero) for the pinouts.
